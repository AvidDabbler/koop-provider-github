# koop-github

## A simple github provider for Koop

## For Developing

## For use within koop

  ```
    # clone Koop:
    git clone git@github.com:Esri/koop.git
    cd koop

    # install depencencies 
    npm install 

    # install koop-github
    npm install https://github.com/chelm/koop-github/taball/master

    # start the koop server
    node server.js 
  ```
